## Features
- Allows you to run a command many times
- Join any multiplayer from anywhere using a command


## Usage
- `/loop <int> <command>` e.g `/loop 20 teleport ~ ~ ~`
- `/join <server address>` e.g `/join play.octanepvp.com` _(coming soon)_
